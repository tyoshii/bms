Baseball Management System
==========================

http://bm-s.info

草野球の試合における成績管理ができるシステムです。

今までの成績管理では、
チームの代表/マネージャー/担当の人がスコア入力をしていたと思いますが、
思ったよりも負荷が高く、それなりに熱意を持った人でも三日坊主になりがちでした。

このシステムでは所属選手一人ひとりがアカウントを持つことが可能となっていて、
自分の成績を自分で入力することが可能となっています。
