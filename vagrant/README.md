# Vagrant

# start vagrant

```
% pwd
/path/to//bms/vagrant

% vagrant box add cent64_up https://github.com/2creatives/vagrant-centos/releases/download/v0.1.0/centos64-x86_64-20131030.box

% vagrant init

% vagrant up && vagrant provision

% vagrant ssh
```
